ThrowError

Formato

throwerror ( número inteiro )
throwerror número inteiro

Description

Causa um erro de execussão. Estes erros podem ser interceptados com Onerror.

Exemplo

onerror errortrap
print "before error"
throwerror 99
print "after error"
end
subroutine errortrap()
print "error " + lasterror + " happened"
end subroutine

will display

before error
error 99 happened
after error
