el:try

Try / Catch / EndTry

Format

try
statement(s)statement or multiple lines of statements
catch
statement(s)statement or multiple lines of statements
end try

Description

Execute the statement(s)statement or multiple lines of statements inside the try. If a runtime error occurs jump to the code in the catch.

See Also

Lasterror, Lasterrorextra, Lasterrorline, Lasterrormessage, Offerror, Onerror, ThrowError, Try / Catch / EndTry
