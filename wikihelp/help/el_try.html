el:try

Try / Catch / EndTry

Σύνταξη

try
κώδικας
catch
κώδικας
end try

Περιγραφή

Οι εντολές μέσα στην try θα εκτελεσθούν. Εάν συμβεί ένα σφάλμα, τότε ο έλεγχος του προγράμματος θα μεταφερθεί στον κώδικα μέσα στην catch.
