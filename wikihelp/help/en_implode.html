en:implode

Implode (Function)

Format

implode ( array_variable )
implode ( array_variable )

returns string_value

Description

Append the elements in an array into a string. Optionally placing the delimiter_expr between the elements. This is functionally the opposite of the Explode function.

Example

dim a$(1)
dim b(1)
a$ = Explode("How now brown cow"," ")
print implode(a$,"-")
print implode(a$)
b = Explode("1,2,3.33,4.44,5.55",",")
print implode(b,", ")
print implode(b)

will display 

How-now-brown-cow
Hownowbrowncow
1, 2, 3.33, 4.44, 5.55
123.334.445.55
