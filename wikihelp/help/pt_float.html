Float

Formato

float ( expression )

Descrição

Convert2 expression num valor decimal.FLOAT converte uma expressão alfanumérica num número decimal. Se a expressão não puder ser convertida será devolvido 0.

Exemplo

a$="5"
print float(a$)/2

mostra 

2.5
