en:toradix

ToRadix

Format

toradix ( number , base )

Description

Changes a number (base 10) to the same number in whatever base defined. The number is returned as a string.

Example

t = 1
print "bases"
print "10,2,5,16,26,36"
do
   print t+",";
   print toradix(t,2)+",";
   print toradix(t,5)+",";
   print toradix(t,16)+",";
   print toradix(t,26)+",";
   print toradix(t,32)
   t*=2
until t >  10000
