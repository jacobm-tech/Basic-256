ToOctal (Function)

Format

ToOctal ( numeric_expr )

returns string_value

Description

This function returns a string containing the Octal (base 8) representation for the number you entered.

Example 

For t = 1 to 10
print ToOctal(t)
next t

Results in 

1
2
3
4
5
6
7
10
11
12
