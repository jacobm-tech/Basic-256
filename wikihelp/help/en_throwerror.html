ThrowError (Statement)

Format

throwerror int_expr
throwerror ( int_expr )

Description

Cause a runtime error to occour. These errors may be trapped with the Onerror statement.

Example

onerror errortrap
print "before error"
throwerror 99
print "after error"
end
subroutine errortrap()
print "error " + lasterror + " happened"
end subroutine

will display

before error
error 99 happened
after error
