Countx

Format

countx ( haystack , regex )

Description

Return the count of the regular expression regex in the string haystack.

Example

print countx("Hello", "[hH]")
print countx("Buffalo buffalo buffalo.","[Bb]uffalo")

will display 

1
3

New to Version

0.9.6.56
