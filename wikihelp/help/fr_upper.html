Upper

Format

upper ( string )

Description

Retourne la chaîne de caractère string avec tous les caractères alphabétiques convertis en majuscule.

See Also

Lower

Example

print upper("La Plage Et Le Soleil !")

Affichera : 

LA PLAGE ET LE SOLEIL !
