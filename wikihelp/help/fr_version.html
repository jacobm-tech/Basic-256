Version

Format

version
version ( )

Description

Retourne un entier représentant le numéro de version de l'environnement BASIC-256 depuis lequel le programme s'exécute.

Exemple

print "Vous utilisez la version " + version()

Affichera quelque chose comme : 

Vous utilisez la version 90932

Disponible depuis la version

0.9.9.32
