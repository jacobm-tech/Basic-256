Imgload (Statement)

Format

imgload x_exprinteger expression representing how far to the right on the 'x' axis, y_exprinteger expression representing how down on the 'y' axis, file_namestring expression containing a file path and name
imgload ( x_exprinteger expression representing how far to the right on the 'x' axis, y_exprinteger expression representing how down on the 'y' axis, file_namestring expression containing a file path and name )
imgload x_exprinteger expression representing how far to the right on the 'x' axis, y_exprinteger expression representing how down on the 'y' axis, scale, file_namestring expression containing a file path and name
imgload ( x_exprinteger expression representing how far to the right on the 'x' axis, y_exprinteger expression representing how down on the 'y' axis, scale, file_namestring expression containing a file path and name )
imgload x_exprinteger expression representing how far to the right on the 'x' axis, y_exprinteger expression representing how down on the 'y' axis, scale, rotation, file_namestring expression containing a file path and name
imgload ( x_exprinteger expression representing how far to the right on the 'x' axis, y_exprinteger expression representing how down on the 'y' axis, scale, rotation, file_namestring expression containing a file path and name )

Description

Load an image or picture from a file and paint it on the Graphics Output Window.
The parameters x_exprinteger expression representing how far to the right on the 'x' axis and y_exprinteger expression representing how down on the 'y' axis represent the location on the screen for the CENTER of the loaded image. This behaviour is different than all of the other graphics statements. The axis of rotation will also be this CENTER point.
The Imgload starement will read in most common image file formats including: BMP (Windows Bitmap), GIF (Graphic Interchange Format),JPG/JPEG (Joint Photographic Experts Group), and PNG (Portable Network Graphics).
Optionally scales size of the loaded image by the defined scale (1=normal size). Also optionally rotates the image by a specified angle around the images center (clockwise in radians).
