el:freedb

FreeDB

Σύνταξη

freedb
freedb ( )

Περιγραφή

Η BASIC256 προσφέρει την δυνατότητα για άνοιγμα πολλών βάσεων δεδομένων ταυτόχρονα. Η συνάρτηση freedb επιστρέφει έναν ελεύθερο αριθμό βάσης δεδομένων που μπορεί να χρησιμοποιηθεί στην επόμενη εντολή DBOpen. Μόλις κλείσει μια βάση δεδομένων, η freedb θα επιστρέψει τον αριθμό της στην λίστα των διαθέσιμων αριθμών βάσεων δεδομένων και μπορεί να ξαναδώσει τον ίδιο αριθμό.

Δες επίσης

DBClose, DBCloseSet, DBExecute, DBFloat, DBInt, DBNull, DBOpen, DBOpenSet, DBRow, DBString. Freedb, Freedbset
