Dir

Formaat

dir ( )
dir ( folder )

Beschrijving

De functie opent de folder om er de namen van de bestanden en folders weer te geven.

Voorbeeld

f$ = dir("c:\")
while f$ <> ""
   print f$
   f$ = dir()
end while
