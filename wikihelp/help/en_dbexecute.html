en:dbexecute

DBExecute

Format

dbexecute SqlStatement
dbexecute ( SqlStatement )
dbexecute DatabaseNumber , SqlStatement
dbexecute ( DatabaseNumber , SqlStatement )

Description

Execute an SQL statement on the open SQLite database file. This statement does not create a record set.

Example

See example of usage on DBOpen page.
