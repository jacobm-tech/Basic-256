NetClose

Formato

netclose
netclose ( )
netclose socket_number
netclose ( socket_number )

Description

Fecha a ligação de rede especificada (socket). Se o socket_number não for especificado o número (0) será usado.

Exemplo

Ver exemplo na página de NetConnect .
