Freefile

Formato

freefile
freefile ( )

Descrição

Basic256 permite vários arquivos para ser aberto em uma única vez. A função freefile devolve um número de ficheiro livre que pode ser usado no próximo Open ou Openb. Quando um ficheiro é fechado, freefile o seu número fica disponível para a lista de números de ficheiro.

Exemplo

# copy one binary file to another
k = 0
source = freefile
openb source,"file.pdf"
dest = freefile
openb dest,"file_copy.pdf"
reset dest
while not eof(source)
   writebyte dest, readbyte(source)
   k++
end while
close dest
close source
print k + " bytes copied."
