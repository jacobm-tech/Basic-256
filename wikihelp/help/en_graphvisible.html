en:graphvisible

GraphVisible

Format

graphvisible ( boolean )
graphvisible boolean

Description

Sets the view option for the “Graphics Output” window. Allows the program to control the checkable option on the View menu of the BASIC-256 IDE.

See Also

EditVisible, OutputVisible

New To Version

0.9.6.65
