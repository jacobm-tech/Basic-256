Spritemove (Statement)

Format

spritemove sprite_number, x_expr, y_expr
spritemove ( sprite_number, x_expr, y_expr )
spritemove sprite_number, x_expr, y_expr, scale_expr
spritemove ( sprite_number, x_expr, y_expr, scale_expr )
spritemove sprite_number, x_expr, y_expr, scale_expr, rotate_expr
spritemove ( sprite_number, x_expr, y_expr, scale_expr, rotate_expr )

Description

Move a sprite from its current position by the specified number of pixels. Motion will be limited to the current screen. Optionally the sprite may be rotated or scaled by defining optional the rrotate_expr and scale_expr parameters. The degree rotate_expr is measured in radians. Rotation and scaling are relative to the previous state of the sprite.
