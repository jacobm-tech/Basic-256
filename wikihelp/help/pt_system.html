System

Formato

system expressão
system ( expressão )

Descrição

Executa um comando do sistema em uma janela de terminal. ATENÇÃO: Esta pode ser uma declaração muito perigosa. Utilize-o apenas se você sabe o que está fazendo.

Esta declaração pode ser desativada por causa de potenciais problemas de segurança do sistema. A disponibilidade pode ser configurada no IDE, indo para o menu Editar> Preferências.

0.9.5h
