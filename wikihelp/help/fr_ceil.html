fr:ceil

Ceil

Format

ceil ( expression )

Description

Retourne l'entier immédiatement inférieur ou égal à une expression.

Voir aussi

Floor

Exemple

print ceil(9.1)
print ceil(-5.4)

fera afficher 

10
-5
