ru:sqr

Sqr

Формат:

sqr( выражение )
sqrt( выражение )

Описание:

Вычисляет квадратный корень выражения. Аналогичный результат можно получить, возведя выражение в степень 1/2. Введена для совместимости с другими версиями BASIC.

Смотри также:

Exp, Log

Пример:

print sqr(4)
print sqr(2)
print 2^(1/2)

Будет напечатано 

2
1.414214
1.414214

Впервые в версии:

0.9.6.51
