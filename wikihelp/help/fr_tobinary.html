ToBinary

Format

ToBinary ( Variable )

Description

Retourne la représentation binaire de la valeur variable sous forme de chaîne de caractères.

Exemple

For t = 0 to 10
print ToBinary(t)
next t

Affichera : 

0
1
10
11
100
101
110
111
1000
1001
1010
