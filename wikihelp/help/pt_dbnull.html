pt:dbnull

DBNull

Formato

dbnull ( ColumnNumber )
dbnull ( DatabaseNumber , ColumnNumber )
dbnull ( DatabaseNumber , DataSetNumber , ColumnNumber )
dbnull ( ColumnName )
dbnull ( DatabaseNumber , ColumnName )
dbnull ( DatabaseNumber , DataSetNumber , ColumnName )

Descrição

Devolve true se o número de coluna especificado ou o nome da linha do conjunto de registos abertos está vazio. Se o campo tiver um valor então false será devolvido.
