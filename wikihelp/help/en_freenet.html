FreeNet (Function)

Format

freenet
freenet ( )

returns positive_integer_value

Description

BASIC256 allows for multiple network connections to be opened at a single time. The freenet function returns a free network socket_number that you can use in your next NetConnect statement. Once a connection is closed, freenet will return that socket_number to the list of available connection numbers and may reissue that number.
