Input

Formato

input expression, stringvariable
input expression, numericvariable
input stringvariable
input numericvarvariable

Descrição

Espera que o utilizador introduza um valor ou texto na janela de saída de texto. Após premir ENTER o que tiver introduzido será associado a stringvar or numericvariable.
A chamada para INPUT pode vir acompanhada de um texto expression.
