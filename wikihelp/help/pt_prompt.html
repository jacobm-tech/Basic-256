pt:prompt

Prompt

Formato

prompt ( message )
prompt ( message , default)

Descrição

Esta função exibe uma caixa de mensagem com uma caixa de entrada e retorna um valor de seqüência de que o usuário digitou na caixa. Você pode definir o valor padrão definindo uma string como argumento segundo.

Exemplo

a$ = prompt("What state do you live in?","KY")
if a$ = "KY" then
   print "Kentucky."
else
   print "Somewhere else"
end if
