en:exp

Exp (Function)

Format

exp ( numeric_expr )

returns float_value

Description

Returns the base of the natural logarithm e (approx: 2.718282) raised to the specified power. This is the inverse of the Log function.

Example

print exp(1)
print exp(log(10))

will print 

2.718282
10
