Redim

Formato

redim numericvariable( integer )
redim stringvariable$( integer )
redim numericvariable( rows , columns )
redim stringvariable$( rows , columns )

Descrição

Redimensiona um array previamente criado, preservando os dados. Se o array é ampliado, os novos elementos serão inicializados com zero ou cadeia alfanumérica vazia. Se uma matriz é reduzida em tamanho, os elementos cortados a partir da extremidade são perdidos.
