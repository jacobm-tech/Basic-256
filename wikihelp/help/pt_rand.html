pt:rand

Rand

Formato

rand
rand()

Descrição

Devolve um número aleatório (decimal) de 0 a 1.

Nota

Para produzir números aleatórios entre outros valores, multiplicar ou adicionar os números apropriados. Por exemplo, para gerar um número inteiro entre 0 e 10, a utilização int (rand * 10).

Exemplo

For n = 1 to 20
Print int(rand*10)+",";
next n

mostra por exemplo 

4,0,7,8,0,5,3,4,5,7,9,1,9,3,6,2,0,9,0,9,
