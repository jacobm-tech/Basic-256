pt:getslice

GetSlice

Formato

getslice(x, y, width, height)

Descrição

Devolve uma cadeia alfanumérica que contém a representação hexadecimal (base 16) do retângulo definido pelos parâmetros. A cadeia é formatada do seguinte modo . Primeiros 4 bytes, largura, os 4 seguintes, altura, seguido de seis bytes por cada pixel (largura*altura).

Exemplo

color 16-1
plot 1,1
color 16^2-1
plot 1,2
color 16^3-1
plot 2,1
color 16^4-1
plot 2,2
print getslice(1, 1, 2, 2)

mostra 

0002000200000f000fff0000ff00ffff
