pt:sound

Sound

Formato

sound frequencia, duração
sound ( frequencia, duração )
sound ( array )
sound array
sound {frequencia1, duração1, frequencia2, duração2, …}

Descrição

Toca um som. A frequência é expressa em Hz e duração é expressa em milissegundos (1000 em um segundo). Um array ou lista contendo frequência e duração também pode ser usada. Isto elimina qualquer clic entre sons quando usados sequencialmente. Suporte de som para sistemas Linux foi adicionado na versão 0.9.5g. Som foi alterado para usar o dispositivo de som padrão no Windows na versão 0.9.5h
