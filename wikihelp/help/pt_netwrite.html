NetWrite

Formato

netwrite string
netwrite ( string )
netwrite socket_number, string
netwrite ( socket_number, string )

Descrição

Envia uma cadeia alfanumérica para a ligação de rede aberta que for especificada. Se o socket_number não for especificado então (0) será assumido por defeito.

Exemplo

Ver exemplo em NetConnect.
