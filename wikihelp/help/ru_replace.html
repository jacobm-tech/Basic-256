ru:replace

Replace

Формат:

replace( строка, что_меняем, на_что_меняем )
replace( строка, что_меняем, на_что_меняем, регистр_не_важен )

Описание:

Возвращает новую строку, где все подстроки что_меняем заменены на подстроки на_что_меняем. Вы можете дополнительно указать булево значение регистра_не_важен = true, если не хотите учитывать регистр символов.

Смотри также:

Replacex

Пример:

a$ = "В лесу родилась елочка, в лесу она росла."
print Replace(a$,"лесу","горах")
print Replace(a$, "ЕлОчКа", "белочка", true)

Будет напечатано 

В горах родилась елочка, в горах она росла.
В лесу родилась белочка, в лесу она росла.
