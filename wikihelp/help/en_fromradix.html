FromRadix (Function)

Format

fromradix ( string_expr, base_expr )

returns integer_value

Description

Converts a string in any base from 2 to 36 into an integer value.

Example

print fromradix("ffef",16)
print fromradix("1001101", 2)
print fromradix("a1z9",36)

displays

65519
77
469125
