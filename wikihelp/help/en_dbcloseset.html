DBCloseSet (Statement)

Format

dbcloseset
dbcloseset ( )
dbcloseset database_number
dbcloseset ( database_number )
dbcloseset database_number , recordset_number
dbcloseset ( database_number, recordset_number )

Description

Close the currently open record set opened by DBOpenSet.

Example

See example of usage on DBOpen page.
