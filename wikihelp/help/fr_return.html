Return

Format

return

Description

L'instruction return est utilisée dans une portion de programme appelée par GOSUB pour retourner le contrôle à l'instruction qui suit le GOSUB. Dans un sous-programme défini par SUBROUTINE ou une fonction définie par FUNCTION, return permet de rendre le contrôle au programme appelant.
