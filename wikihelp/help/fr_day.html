Day

Format

day
day()

Description

Renvoie le jour du mois actuel (1-31) d'après l'horloge système.

Exemple

print "Aujourd'hui nous sommes le ";
print day() + "/" + (month() + 1) + "/" + year()

affichera par exemple : 

Aujourd'hui nous sommes le 1/5/2013
