Line

Format

line x0, y0, x1, y1
line(x0, y0, x1, y1)

Description

Trace une ligne reliant les points x0,y0 et x1, y1.

Exemple

color white
rect 0,0,300,300
color black
line 50,50,200,200
line 100,200,200,200
line 100,200,50,50
