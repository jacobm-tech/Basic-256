en:editvisible

EditVisible (Statement)

Format

editvisible ( boolean_exprboolean (true/false) expression )
editvisible boolean_exprboolean (true/false) expression

Description

Sets the view option for the “Edit” portion of the main window, this is where the program is displayed and edited.

Example

editvisible 0
pause 2
editvisible 1

See Also

EditVisible, Include, Ostype, System, Version
