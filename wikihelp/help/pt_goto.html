Goto

Formato

goto marcador

Description

Faz saltar a execussão para o marcador indicado.

See Also

Gosub

Exemplo

print "I";
goto skipit
print " don't";
skipit: #
print " want cookies."

imprime 

I want cookies.

Notas

As of version 0.9.9.2 Goto, Gosub, and labels can not be used in Function and Subroutine definitions.
