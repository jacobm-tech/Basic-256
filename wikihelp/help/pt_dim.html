pt:dim

Dim

Formato

dim numericvariable ( inteiro(n) )
dim stringvariable$ ( inteiro(n) )
dim numericvariable ( linha ,coluna )
dim stringvariable$ ( linha ,coluna)

Descrição

Cria um array (vector) unidimensional de com n elementos ou bidimensional linha x coluna . Os arrays podem conter números ou cadeias alfanuméricas “palavras”.
O primeiro elemento tem ordem 0 (zero). A ordem dos elementos varia de 0 até n-1

Ver também

Dim, Redim

Exemplo 1

dim z(5)
z = {1, 2, 3, 4, 5}
print z[0] + " " + z[4]

will print 

1 5

Exemplo 2

dim c$(4)
c$ = {"cow", "brow", "how", "now"}
print c$[2] + " " + c$[3] + " ";
print c$[1] + " " + c$[0] + "?"

will print 

how now brown cow?
