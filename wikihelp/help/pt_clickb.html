Clickb

Formato

clickb
clickb ( )

Description

Devolve os botões do rato que o utilizador premiu por último (se estiver sobre a janela gráfica).Devolve 0 se nenhum botão for premido.Se vários botões forem premidos devolve a soma.

Valores devolvidos
ValorBotão premido
0Nenhum
1Esquerdo
2Direito
4Centro

Ver Também

Clickclear, Clickx, Clicky, Mouseb, Mousex, Mousey

Exemplo

# apaga os valores preexistentes
clickclear
# aguarda que o utilizador pressione o botão
print "click mouse on the graphics output"
while clickb = 0
  pause .01
endwhile
# revela one foi premido
print "The user clicked at (" + clickx + "," + clicky + ")"
