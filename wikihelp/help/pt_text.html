Text

Formato

text x, y, string
text ( x, y, string )

Descrição

Paints a text string on the Graphics Output Window at x, y using the current color and font. Exibe texto (cadeia alfanumérica) na janela gráfica na coordenada x, y e na cor e tipo presentes

Exemplo

color grey
rect 0,0,graphwidth,graphheight
color red
font "Times New Roman",18,50
text 10,100,"This is Times New Roman"
color darkgreen
font "Tahoma",28,100
text 10,200,"This is BOLD!"

Will draw.
