ToRadix

Format

toradix ( number , base )

Description

Retourne la représentation en base base de la valeur number sous forme de chaîne de caractères.

Exemple

t = 1
print "bases"
print "10,2,5,16,26,36"
do
   print t+",";
   print toradix(t,2)+",";
   print toradix(t,5)+",";
   print toradix(t,16)+",";
   print toradix(t,26)+",";
   print toradix(t,32)
   t*=2
until t >  10000
