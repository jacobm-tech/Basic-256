pt:sqr

Sqr or Sqrt

Formato

sqr ( n )
sqrt ( n )

Description

Devolve a raiz quadrada de n.
Nota: também é possível usar potências fracionárias sqr(n) = n^(1/2). Para o caso de raiz cúbica n^(1/3) ou superior será esta a forma de as obter.

Exemplo

print sqr(49)
print sqrt(25)
print 1000^(1/3)

will print 

7
5
10

Introduzido

0.9.6.51
