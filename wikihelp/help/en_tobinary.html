en:tobinary

FromBinary (Function)

Format

ToBinary ( numeric_exprnumeric expression )

returns string_valuea string of zero or more characters

Description

Converts a number to its a string containing a binary value of ones and zeros.

Example 

For t = 0 to 10
print ToBinary(t)
next t

Results in 

0
1
10
11
100
101
110
111
1000
1001
1010

FromBinary, FromHex, FromOctal, FromRadix,ToBinary, ToHex, ToOctal, ToRadix
