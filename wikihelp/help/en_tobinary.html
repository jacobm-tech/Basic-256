en:tobinary

Format

ToBinary ( Variable )

Description

Converts a number to its binary value.

Example 

For t = 0 to 10
print ToBinary(t)
next t

Results in 

0
1
10
11
100
101
110
111
1000
1001
1010

See also frombinary
