Chr

Format

chr ( expression )

Description

Converts the integer expression into a single character string expression with the ASCII value of the number. See asc for a complete ASCII character conversion chart.

See Also

Asc

Example

print chr(66)+chr(111)+chr(111)+chr(33)

will print 

Boo!
