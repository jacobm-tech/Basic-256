PortOut

Formato

portout ( io port number, value )

Descrição

Escreve uma porta de entrada e saída do sistema e devolve um valor de (0-255). Ler e escrever em I/O ports pode ser perigoso e ter resultados imprevistos. Esta faculdade pode ser desligada devido a possíveis riscos, para tal vá ao menu Edit>Preferences menu. Esta funcionalidade apenas está disponível nas portas windows.

São necessárias permissões de administrador para instalar a driver inpout32 necessária para esta funcionalidade operar. Quando a driver for instalada as PortIn/PortOut funcionarão.

Ver também

PortIn

Exemplo

for y = 0 to 255
   portout 0x378,y
   print y + " " + portin(0x379)
   pause .1
next

Escreve os números 0-255 na primeira porta de impressora (porta em paralelo standard) e mostrará o valor do registo de estado.
