Goto

Format

goto label

Description

Jumps to the specified label.

See Also

Gosub

Example

print "I";
goto skipit
print " don't";
skipit: #
print " want cookies."

will print 

I want cookies.

Notes

As of version 0.9.9.2 Goto, Gosub, and labels can not be used in Function and Subroutine definitions.
