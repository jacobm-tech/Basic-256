pt:implode

Implode

Formato

implode ( arrayvariable )
implode ( arrayvariable , delimitador )

Descrição

Concatena os elementos de um array numa única cadeia alfanumérica. Como opção pode-se colocar um delimitadior entre os elementos. Esta função faz o posto de Explode .

Exemplo

dim a$(1)
dim b(1)
a$ = Explode("How now brown cow"," ")
print implode(a$,"-")
print implode(a$)
b = Explode("1,2,3.33,4.44,5.55",",")
print implode(b,", ")
print implode(b)

mostra 

How-now-brown-cow
Hownowbrowncow
1, 2, 3.33, 4.44, 5.55
123.334.445.55
