el:dir

Dir

Σύνταξη

dir ( )
dir ( ΔιαδρομήΑρχείου )

Περιγραφή

Ανοίγει τον φάκελο που προσδιορίζεται από την ΔιαδρομήΑρχείου και ανακτά τα ονόματα των αρχείων και των φακέλων που περιέχει.

Παράδειγμα

f$ = dir("c:\")
while f$ <> ""
   print f$
   f$ = dir()
end while
