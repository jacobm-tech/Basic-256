FreeDB (Function)

Format

freedb
freedb ( )

returns positive_integer_value

Description

BASIC256 allows for multiple databases to be opened at a single time. The freedb function returns a free database_number that you can use in your next DBOpen statement. Once a database is closed, freedb will return that database number to the list of available database numbers and may reissue that number.
