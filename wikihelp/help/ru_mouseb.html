ru:mouseb

Mouseb

Формат:

mouseb
mouseb()

Описание:

Возвращает код нажатой кнопки мыши (курсор должен быть над окном для вывода графики). Возвращает 0, если нажатий не было. Если несколько кнопок нажато одновременно, возвращается сумма значений нажатых кнопок. === Возвращаемые значения ===|Значение|Нажата клавиша|

0Никакая
1Левая
2Правая
4Средняя

Смотри также:

Clickb, Clickclear, Clickx, Clicky, Mousex, Mousey

0.9.4d
