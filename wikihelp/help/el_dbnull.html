el:dbnull

DBNull

Σύνταξη

dbnull ( ΑριθμόςΣτήλης )
dbnull ( ΑριθμόςΒάσηςΔεδομένων , ΑριθμόςΣτήλης )
dbnull ( ΑριθμόςΒάσηςΔεδομένων , ΑριθμόςΠίνακα , ΑριθμόςΣτήλης )
dbnull ( ΌνομαΣτήλης )
dbnull ( ΑριθμόςΒάσηςΔεδομένων , ΌνομαΣτήλης )
dbnull ( ΑριθμόςΒάσηςΔεδομένων , ΑριθμόςΠίνακα , ΌνομαΣτήλης )

Περιγραφή

Επιστρέφει true εάν το συγκεκριμένο πεδίο του δοσμένου αριθμού ή ονόματος στήλης της τρέσουσας σειράς του ανοιγμένου πίνακα έχει τιμή NULL. Εάν το πεδίο δεν είναι κενό, τότε θα επιστραφεί false.

Παράδειγμα

Δες πως μπορείς να χρησιμοποιήσεις την εντολή στην σελίδα DBOpen.
