pt:dbclose

DBClose

Formato

dbclose
dbclose ( )
dbclose DatabaseNumber
dbclose ( DatabaseNumber )

Descrição

Fecha um ficheiro SQLite (base de dados).As ligações da base de dados são numeradas de 0 a 7. Se o número for omitido então o ficheiro #0 será fechado.

Exemplo

Ver o comando DBOpen.
