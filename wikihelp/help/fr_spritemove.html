Spritemove

Format

spritemove numerosprite, dx, dy
spritemove ( numerosprite, dx, dy )

Description

Déplace le “sprite” numerosprite depuis sa position actuelle du nombre spécifié de pixels dx, dy. Le déplacement est limité à l'écran graphique affiché.
