FreeNet

Format

freenet
freenet ( )

Description

Avec BASIC256 il est possible d'ouvrir plusieurs connexions réseau même temps (voir NetConnect). La fonction freenet retourne un numéro encore non utilisé pour l'ouverture d'une connexion réseau avec NetConnect. Lorsque la connexion réseau sera ensuite fermée, le numéro qui était associé sera libéré et pourra de nouveau être retourné par freenet.
