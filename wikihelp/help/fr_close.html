Close

Format

close
close ( )
close filenumber
close ( filenumber )

Description

Ferme un fichier ouvert. Si aucun fichier ne correspond à ce numéro, la commande n'a aucun effet. Si aucun numéro de fichier n'est spécifié, le numéro de fichier utilisé sera le zéro.
