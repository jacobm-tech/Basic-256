fr:currentdir

Currentdir

Format

currentdir
currentdir ( )

Description

Retourne le chemin complet du répertoire courant. Pour toutes les OS (y compris Windows) un anti-slash (/) sépare les répertoires du chemin.

Disponible à partir de la version

0.9.6r
