en:instr

Instr (Function)

Format

instr ( haystack_exprstring expression to search in , needle_exprstring expression to search for )
instr ( haystack_exprstring expression to search in , needle_exprstring expression to search for , start_exprnumeric expression representing start of for loop or starting location in string )
instr ( haystack_exprstring expression to search in , needle_exprstring expression to search for , start_exprnumeric expression representing start of for loop or starting location in string , boolean_exprboolean (true/false) expression)

returns positive_integer_valuea non-negative whole number

Description

Check to see if the string needle_exprstring expression to search for is contained in the string haystack_exprstring expression to search in. If it is, then this function will return the index of starting character of the first place where needle_exprstring expression to search for occurs. Otherwise, this function will return 0. You may also specify an optional starting location for the search to begin start_exprnumeric expression representing start of for loop or starting location in string and a boolean_exprboolean (true/false) expression to specify that the search will treat upper and lower case letters the same.

Note

String indices begin at 1.

Example

print instr("Hello", "lo")
print instr("101,222,333",",",5)

will display 

4
8
