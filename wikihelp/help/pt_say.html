Say

Formato

say expression
say ( expression )

Description

Usa o servoço TTS “Text to speech” para dizer expression. Em LINUX é necessário instalar FLite ou eSpeak. Em Windows é usada a SAPI do sistema. 

# Statements to set default language to 'fr' on Ubuntu 10.10
# Your system may be different
cd /usr/share/espeak-data/voices
sudo cp fr default

Introduzido na versão

0.9.4
