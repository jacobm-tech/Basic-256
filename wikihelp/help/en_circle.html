Circle (Statement)

Format

circle x_expr, y_expr, r_expr
circle ( x_expr, y_expr, r_expr )

Description

Draws a circle centered at x,y with a radius r using the current pen and brush colors.

Example

clg
color red
circle 75,75,50
penwidth 5
color orange, yellow
circle 150,150,75
penwidth 10
color blue, clear
circle 225,225,50

draws
