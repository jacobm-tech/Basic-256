nl:size

Size

Formaat

size
size()
size(filenumber)

Beschrijving

Deze functie geeft weer hoe groot het bestand is in bytes1). Geef je geen bestandnummer in, dan wordt de functie uitgevoerd op bestandnummer 0, als ze bestaat natuurlijk.

Voorbeeld

print currentdir
open (1,"Foo.txt")
print size(1) 
close 1

geeft dan 

/home/guest # werkdirectory :-) 
40 # dit bestand is dus 40 bytes lang

Geef je geen bestandnummer in, dan probeert het programma 0. Maar aangezien er geen bestand geopend is voor 0, krijg je FOUT op lijn 3: File not open.
