pt:imgsave

ImgSave

Formato

imgsave filename
imgsave filename, type
imgsave ( filename )
imgsave ( filename, type )

Descrição

Guarda a imagem da janela grafica para um ficheiro de imagem.
Por defeito é assumido o formato (PNG). Como opção há também a possibilidade de gravar em “BMP”, “JPG”, “JPEG”.

Por vezes as imagens serão guardadas como “fundo negro” onde a janela não foi desenhada.Isto deve-se à instrução CLG establece como cor de fundo a cor “clear” (transparente).Quando a imagem é guardada o transparente passa a negro. Pode alterar-se a instrução clg do seguinte modo: 

color white
rect 0, 0, graphwidth, graphheight

Exemplo

color white
rect 0, 0, graphwidth, graphheight
for t = 0 to 100
   color rand()*256, rand()* 256, rand()*256
   rect rand()*graphwidth, rand()*graphheight,rand()*graphwidth, rand()*graphheight
next t
imgsave "testimgsave1.png"
imgsave "testimgsave2.jpg", "jpg"
