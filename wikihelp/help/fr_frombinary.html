FromBinary

Format

frombinary ( string )

Description

Cette fonction retourne l'entier correspondant à la valeur binaire (base 2) passée en paramètre comme chaîne de caractère string.

Exemple

print frombinary("11")
print frombinary("1001101")

Affichera

3
77

Disponible depuis la version

0.9.9.45
