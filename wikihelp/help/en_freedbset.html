FreeDBSet (Function)

Format

freedbset ( database_number )

returns positive_integer_value

Description

BASIC256 allows for multiple databases with multiple result sets to be opened at a single time. The freedbset function returns a free recordset_number for an open database connection that you can use in your next DBOpenSet statement. Once a database set is closed, freedbset will return that file number to the list of available database set numbers and may reissue that number.
