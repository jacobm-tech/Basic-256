en:dbnull

DBNull

Format

dbnull ( ColumnNumber )
dbnull ( DatabaseNumber , ColumnNumber )
dbnull ( DatabaseNumber , DataSetNumber , ColumnNumber )
dbnull ( ColumnName )
dbnull ( DatabaseNumber , ColumnName )
dbnull ( DatabaseNumber , DataSetNumber , ColumnName )

Description

Return a true if the specified column number or name of the current row of the open recordset is a NULL vale. If the field contains a value a false will be returned.

Example

See example of usage on DBOpen page.
