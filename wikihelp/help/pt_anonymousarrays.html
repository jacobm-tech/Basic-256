pt:anonymousarrays

Arrays inomidados

Um array inominado é um conjunto de valores numéricos ou um conjunto de valores de cadeia, separados por vírgulas e entre chaves {}. Estes arrays anónimos podem ser utilizados para atribuir rapidamente um grupo de valores a um array (com nome). Se o array inominado é atribuido a um array menor este será automaticamente redimensionados (Redim) para o comprimento correto.

Arrays inomidados também podem ser usados em Sound, Poly, e Stamp em lugar de um array denominado.

Exemplo

dim myarray(4)
myarray = {1, 2, 3, 4}
dim words$(1)
words$ = {"how","now","brown","cow"}
for n = 0 to words$[?]-1
 print words$[n]
next n
