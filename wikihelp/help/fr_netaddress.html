fr:netaddress

NetAddress

Format

netaddress
netaddress ( )

Description

Retourne une chaîne de caractères contenant l'adresse IPV4 de la machine sur le réseau local.

Exemple

print netaddress()

Affichera quelque chose comme : 

192.168.150.10
