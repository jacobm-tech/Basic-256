DBString (Function)

Format

dbstring ( numeric_expr )
dbstring ( database_number , numeric_expr )
dbstring ( database_number , recordset_number , numeric_expr )
dbstring ( string_expr )
dbstring ( database_number , string_expr )
dbstring ( database_number , recordset_number , string_expr )

returns string_value

Description

Return a string from the specified column number or name of the current row of the open recordset. If the field is a NULL value the empty string ”” will be returned. NULL may be tested for by using the DBNull function.

Example

See example of usage on DBOpen page.
