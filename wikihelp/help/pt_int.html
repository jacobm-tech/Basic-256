pt:int

Int

Formato

int ( expression )

Descrição

Converte expression para um número inteiro. INT converte uma expressão alfanumérica ou um número decimal. Se a expressão não puder ser convertica será devolvido 0.

Exemplo

a$="123"
print int(a$)
print int(123.4)

mostra 

123
123
