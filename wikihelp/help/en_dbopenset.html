en:dbopenset

DBOpenset (Statement)

Format

dbopenset sql_statementstring expression containing an SQL statement
dbopenset ( sql_statementstring expression containing an SQL statement )
dbopenset database_numberinteger expression representing one of the current database handles , sql_statementstring expression containing an SQL statement
dbopenset ( database_numberinteger expression representing one of the current database handles , sql_statementstring expression containing an SQL statement )
dbopenset database_numberinteger expression representing one of the current database handles , recordset_numberinteger expression representing one of the current database recordset handles , sql_statementstring expression containing an SQL statement
dbopenset ( database_numberinteger expression representing one of the current database handles , recordset_numberinteger expression representing one of the current database recordset handles , sql_statementstring expression containing an SQL statement )

Description

Perform an SQL statement and create a record set so that the program may loop through and use the results.

Example

See example of usage on DBOpen page.
