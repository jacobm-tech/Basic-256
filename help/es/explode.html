Explode

Formato

VariableArraydeCadenas$ = explode ( cadena , delimitador )
VariableArraydeCadenas$ = explode ( cadena , delimitador , NoDistinguirMayúsculas )
VariableArray = explode ( cadena , delimitador )
VariableArray = explode ( cadena , delimitador , NoDistinguirMayúsculas )

Descripción

Divide cadena en subcadenas, determinadas por el delimitador. Las subcadenas se almacenarán bien en un array de cadenas, o bien en un array numérico, según como se asigne la función. El array será redimensionado al tamaño exacto para almacenar todas las subcadenas. Puede especificarse un valor booleano opcional NoDistinguirMayúscula para indicar que la búsqueda tratará la mayúscula y la minúscula como una misma letra.

Ejemplo 

# dividir con el espacio como delimitador
a$ = "We all live in a yellow submarine."
print a$
w$ = explode(a$," ")
for t = 0 to w$[?]-1
   print "w$["+t+"]=" + w$[t]
next t

# dividir con A o a como delimitador
a$ = "klj;lkjalkjAlkj;"
print a$
w$ = explode(a$,"A",true)
for t = 0 to w$[?]-1
   print "w$["+t+"]=" + w$[t]
next t

# dividir, con la coma como delimitador
a$="1,2,3,77,foo,9.987,6.45"
print a$
n = explode(a$,",")
for t = 0 to n[?]-1
   print "n["+t+"]=" + n[t]
next t
