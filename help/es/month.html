month
Index

Month

Formato

month
month()

Descripción

Devuelve el mes actual de acuerdo con el reloj del sistema. Enero es 0, febrero es 1,...diciembre es 11.

Ejemplo 

cls
dim n$(12)
n$ = {"ene", "feb", "mar", "abr", "may", "jun", "jul", "ago", "sep", "oct", "nov", "dic"}
print day + "-" + n$[month] + "-" + year
el día de Año Nuevo mostraría: 
1-ene-2010
