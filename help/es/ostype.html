Index

Ostype

Formato

ostype ( )

Descripción

Devuelve un número que representa el sistema operativo/entorno de software para el que fue compilado el archivo ejecutable de BASIC256.
Valores devueltos
ValorTipo
0Windows
1Linux
2Macintosh

Ejemplo 

print "Estás usando un equipo con ";
if ostype() = 0 then
   print "windows";
else
   print "unix/linux";
end if
mostrará algo como esto: 
Estás usando un equipo con unix/linux.
