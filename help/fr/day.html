day
Index

Day

Format

day
day ( )

Description

Retourne le jour du mois actuel (1-31) d'après l'horloge système.

Voir aussi

Hour, Minute, Month, Second, Year

Exemple 

print "Aujourd'hui nous sommes le ";
print day + "/" + (month + 1) + "/" + year
affichera| 
Aujourd'hui nous sommes le 12/09/2010
