chr
Index

Chr

Format

chr ( expression )

Description

Converti l'entier expression en le caractère correspondant en code ASCII. Voir la page consacrée à asc pour un tableau de converson des caractères ASCII.

Voir aussi

Asc

Exemple 

print chr(66)+chr(111)+chr(111)+chr(33)
affichera 
Boo!
