count
Index

Count

Формат:

count( строка, искомая_строка )
count( строка, искомое, регистр_не_важен )

Описание:

Возвращает количество строк искомая_строка в переменной или константе строка. Вы также можете указать дополнительное булево значение регистр_не_важен = true, если хотите не учитывать регистр букв.

Смотри также:

Countx

Пример: 

print count("Привет", "ве")
print count("Буйвол, буйвол, БуЙвОл.","БУЙВОЛ",true)

3
