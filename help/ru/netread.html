netread
Index

NetRead

Формат:

netread
netread( )
netread( номер_сокета )

Описание:

Читает данные из сетевого соединения и возвращает их в виде строки. Эта функция ждет, пока происходит получение данных. Если номер_сокета, используется нулевой (0) номер.

Смотри также:

NetAddress, NetClose, NetConnect, NetData, NetListen, NetWrite

Пример использования смотри на странице NetConnect.
