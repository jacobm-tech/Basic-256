Sound

Формат:

sound частота, длительность
sound( частота, длительность )
sound( массив )
sound массив
sound { частота1, длительность1, частота2, длительность2, ... }

Описание:

Воспроизводит звук через динамики компьютера. Частота задается в герцах (Hz), длительность - в миллисекундах (1/1000 секунды). Функции можно передать массив или список состоящий из пар частота, длительность. Передача массива или списка помогает избежать щелчков между двумя соседними звуками воспроизводимыми последовательно. Поддержка звука для LINUX была добавлена в версии 0.9.5g. Функция Sound была изменена на использование стандартного звукового устройства в WindowsTM в версии 0.9.5h
