countx
Index

Countx

Формат:

countx( строка, рег_выражение )

Описание:

Возвращает количество строк (заданных регулярным выражением) рег_выражение в строке строка.

Смотри также:

Count

Пример: 

print countx("Привет", "[пП]")
print countx("Буйвол, буйвол, буйвол.","[Бб]уйвол")
Будет напечатано 
1
3

0.9.6.55
