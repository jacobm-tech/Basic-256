MD5

Формат:

md5( строка )

Описание:

Возвращает MD5 хэш строки, заданной аргументом строка. Эта функция была получена из RSA Data Security, Inc MD5 Message-Digest Algorithm.

Пример: 

print MD5("Something")
print MD5("something")
