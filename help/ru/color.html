color
Index

Color

Формат:

color имя_цвета
color( имя_цвета )
color красный, синий, зеленый
color( красный, синий, зеленый )
color rgb_значение
color( rgb_значение )

Описание:

Устанавливает текущий цвет для операций рисования в значение имя_цвета; или в RGB цвет из компонент красный, синий, зеленый; или в rgb_значение. Есть специальный цвет с именем CLEAR и значением -1. Если текущий цвет установлен в это значение, то пикселы или фигуры, рисуемые этим цветом будут стирать пикселы окна графического вывода делая их прозрачными. Это особенно удобно при создании спрайтов через команду Spriteslice.

Название цвета и их RGB значения

|Название цвета|Вид цвета|RGB значение|
black 0, 0, 0
white 255, 255, 255
red 255, 0, 0
darkred 128, 0, 0
green 0, 255, 0
darkgreen 0, 128, 0
blue 0, 0, 255
darkblue 0, 0, 128
cyan 0, 255, 255
darkcyan 0, 128, 128
purple 255, 0, 255
darkpurple 128, 0, 128
yellow 255, 255, 0
+
