eof
Index

Eof

Формат:

eof
eof()
eof( номер_файла )

Описание:

Возвращает двоичный флаг (истина/ложь) в качестве признака конца чтения файла EOF (на англ. End Of File - конец файла). Если номер_файла не указан, используется значение нуль (0).

Смотри также:

Changedir, Close, Currentdir, Exists, Kill, Open, Read, Readline, Reset, Seek, Size, Write, Writeline
