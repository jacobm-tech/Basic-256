int
Index

Int

Formaat

int ( expression )

Beschrijving

De functie converteer de expression naar een geheel getal. Zowel een string als een decimaal getal worden dan geconverteerd.Lukt de conversie niet, dan wordt 0 teruggestuurd. Deze functie is vooral nuttig samen met de rand functie.

Voorbeeld 

a= 4.53
print a
print int(a)
geeft dan 
4.53
4
