countx
Index

Countx

Formaat

count ( hooiberg , regex )

Beschrijving

De functie geeft weer hoeveel keer de reguliere expressie regex in de hooiberg werd gevonden.

Voorbeeld 

print countx("Hello", "[hH]")
print countx("Buffalo buffalo buffalo.","[Bb]uffalo")
toont 
1
3
