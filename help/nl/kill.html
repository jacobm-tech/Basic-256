Kill

Formaat

kill ( uitdrukking )

Beschrijving

Deze functie verwijdert het bestand zoals in uitdrukking van het system .
Indien het bestand niet bestaat, krijg je een foutmelding.

Zie ook

Close, Eof, Open, Read, Readline, Reset, Seek, Size, Write, Writeline

Voorbeeld 

file$="failed.wav"
print currentdir ( )
if  exists (file$) then 
kill (file$)
else 
print "bestand niet gevonden, niets gewist"
end if
