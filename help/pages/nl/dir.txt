===== Dir =====

==== Formaat ====
**dir** ( )\\
**dir** ( //folder// )

==== Beschrijving ====
De functie opent de //folder// om er de namen van de bestanden en folders weer te geven. 

==== Voorbeeld ====
<code>
f$ = dir("c:\")
while f$ <> ""
   print f$
   f$ = dir()
end while
</code>

kan dan bvb dit geven ( op een windows OS )
<code>
$Recycle.Bin
autoexec.bat
Backup
Boot
bootmgr
Documents and Settings
IO.SYS
MSDOS.SYS
MSOCache
pagefile.sys
Program Files
ProgramData
System Volume Information
temp
Users
Windows
</code>
