===== DBExecute =====

==== Formato ====
**dbexecute** //InstrucciónSQL//\\
**dbexecute** ( //InstrucciónSQL// )

==== Descripción ====
Ejecuta una instrucción SQL en el archivo de base de datos SQLite abierto. Esta instrucción no crea un //recordset//.

==== Ejemplo ====
Ver ejemplo de uso en la página de [[DBOpen|DBOpen]].
