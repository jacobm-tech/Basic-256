===== Dir =====

==== Formato ====
**dir** ( )\\
**dir** ( //carpeta// )

==== Descripción ====
Abre una //carpeta// para obtener los nombres de los archivos o carpetas que contenga.

==== Ejemplo ====
<code>
f$ = dir("c:\")
while f$ <> ""
   print f$
   f$ = dir()
end while
</code>
mostrará algo similar a esto:
<code>
$Recycle.Bin
autoexec.bat
Backup
Boot
bootmgr
Documents and Settings
IO.SYS
MSDOS.SYS
MSOCache
pagefile.sys
Program Files
ProgramData
System Volume Information
temp
Users
Windows
</code>
