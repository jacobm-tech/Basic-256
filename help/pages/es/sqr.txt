===== Sqr o Sqrt =====

==== Formato ====

**sqr** ( //expresión// )\\
**sqrt** ( //expresión// )

==== Descripción ====

Devuelve la raiz cuadrada de // expresión//.

==== Ejemplo ====

<code>
print sqr(2)
print sqrt(10)
</code>
mostrará:
<code>
1.414214
3.162278
</code>
