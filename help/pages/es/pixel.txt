===== Pixel =====
==== Formato ====
**pixel** (//x//, //y// )

==== Descripción ====
Devuelve el valor RGB del pixel situado en las coordenadas //x//,//y//. Si el pixel no ha sido modificado desde la última instrucción [[clg|Clg]] o fue pintado con el color CLEAR, se devuelve el valor -1.

==== Ver también ====
[[rgb|Rgb]]

==== Disponible desde la versión ==== 
0.9.5m
