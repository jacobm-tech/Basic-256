===== DBopenset =====

==== Formato ====
**dbopenset** //InstrucciónSQL//\\
**dbopenset** ( //InstrucciónSQL// )

==== Descripción ====
Ejecuta una instrucción SQL y crea un //recordset// que el programa puede utilizar.

==== Ejemplo ====
Ver ejemplo de uso en la página de [[DBOpen|DBOpen]].
