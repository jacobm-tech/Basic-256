===== Reset =====
==== Formato ====
**reset**\\
**reset()**\\
**reset**(//NúmerodeArchivo//)
==== Descripción ====
Borra el contenido de un archivo abierto. Todos los datos almacenados en él se pierden. Si el número de archivo no se especifica, se usa el número cero (0).
