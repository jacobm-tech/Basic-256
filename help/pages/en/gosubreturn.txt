===== Gosub / Return =====
==== Format ====
**gosub** //label//\\
\\
label:\\
//statement(s)//\\
**return**

==== Description ====
Jumps to the specified label.  Upon encountering a RETURN command, program execution will continue at the line following the GOSUB.  GOSUB commands may be nested.
