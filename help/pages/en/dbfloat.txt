===== DBFloat =====

==== Format ====
**dbfloat** ( //ColumnNumber// )

==== Description ====
Return a floating point (decimal value) from the specified column of the current row of the open recordset.

==== Example ====
See example of usage on [[DBOpen|DBOpen]] page.
