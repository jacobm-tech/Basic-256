===== Dir =====

==== Format ====
**dir** ( )\\
**dir** ( //folder// )

==== Description ====
Open a //folder// to retrieve the names the files or folders that are contained in it.

==== Example ====
<code>
f$ = dir("c:\")
while f$ <> ""
   print f$
   f$ = dir()
end while
</code>
will display something like
<code>
$Recycle.Bin
autoexec.bat
Backup
Boot
bootmgr
Documents and Settings
IO.SYS
MSDOS.SYS
MSOCache
pagefile.sys
Program Files
ProgramData
System Volume Information
temp
Users
Windows
</code>
