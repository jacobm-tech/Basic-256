===== Volume =====
==== Format ====
**volume** //level//\\
**volume** ( //level// )
==== Description ====
Adjust the volume of the notes played with the [[sound|Sound]] command.  Volume //levels// must be numeric values from 0 to 10.  The default volume is 5.

[[sound|Sound]]
