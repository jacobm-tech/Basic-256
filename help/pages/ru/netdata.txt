==== NetData ====
=== Формат: ===
**netdata**\\
**netdata**( )\\
**netdata** номер_сокета\\
**netdata**( номер_сокета )
=== Описание: ===
Возвращает значение true (истина, числовое значение 1), если имеются данные, которые можно прочитать функцией [[netread|NetRead]], в противном случае возвращает false (ложь, значение 0). Если //номер_сокета//, используется нулевой (0) номер.

Пример использования смотри на странице [[netconnect|NetConnect]].
