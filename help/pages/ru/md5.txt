==== MD5 ====
=== Формат: ===
**md5**( строка )
=== Описание: ===
Возвращает MD5 хэш строки, заданной аргументом //строка//. Эта функция была получена из RSA Data Security, Inc MD5 Message-Digest Algorithm.
=== Пример: ===

<code>
print MD5("Something")
print MD5("something")
</code>
Будет напечатано
<code>
73f9977556584a369800e775b48f3dbe
437b930db84b8079c2dd804a71936b5f
</code>
