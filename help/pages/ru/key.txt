==== Key ====
=== Формат: ===
**key**\\
**key**()
=== Описание: ===
Немедленно возвращает целое число, соответствующее нажатой на клавиатуре клавиши.
=== Пример: ===
Код
<code>
if key = 47 then print key
</code>
 не будет работать, поскольку функция key вызывается дважды и каждый раз будет давать разные значения. Вот правильное использование функции key: 
<code>
a = key
if a = 47 then print a
</code>
