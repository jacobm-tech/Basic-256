==== Синтаксис программы ====
Программа на BASIC-256 состоит из списка операторов (команд), записанных по одному на строке и выполняющихся по очереди. Например:
<code>
print "1я строка"
print "2я строка"
</code>
выведет
<code>
1я строка
2я строка
</code>
Кроме того, существуют операторы, позволяющие изменять последовательный порядок выполнения команд:
  * [[goto|Goto]] (переход по метке),
  * [[ifthen|If / Then]] (проверка условия),
  * [[fornext|For / Next]], [[dountil|Do / Until]], [[whileendwhile|While / End While]] (циклы),
  * [[gosubreturn|Gosub / Return]] (подпрограммы)
.
