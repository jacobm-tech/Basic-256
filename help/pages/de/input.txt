===== Input =====
==== Format ====
input //Zeichenkette//, //Stringvariable//
==== Beschreibung ====
Gibt //Zeichenkette// aus und erwartet eine Texteingabe im Textfenster. Die Eingabe mu&szlig; mit der Eingabe-Taste abgeschlossen werden. Die eingelesene Zeile wird in Variable //Stringvariable// gespeichert.
==== Beispiel ====
' ' input "Bitte gib Deinen Namen ein: ", name$' '
