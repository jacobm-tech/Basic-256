===== Lower =====
==== Format ====
**lower**( //string//)

==== Description ====
Returns //string// with all alphabetic characters converted to lower case.
==== See Also ====
[[upper|Upper]]
<code>
print lower("BlUe!")
</code>
will display
<code>
blue!
</code>
