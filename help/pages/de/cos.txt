===== Cos =====
==== Format ====
**cos** ( //Ausdruck// )
==== Beschreibung ====
Berechnet den Kosinus von //Ausdruck//. //Ausdruck// muss ein Bogenma&szlig; sein.
==== Note ====
Die cos-Funktion liefert kein besonders genaues Ergebnis.
==== Siehe auch ====
[[sin|Sin]], [[tan|Tan]]
==== Beispiel ====
<code>
c = cos( 2*PI*deg/360)
</code>
