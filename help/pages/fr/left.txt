===== Left =====
==== Format ====
left( //string//, //length//)
==== Description ====
Returns a portion of the specified //string//, starting from the first character on the left and continuing for //length// characters.
==== See Also ====
[[mid|Mid]], [[right|Right]]
==== Example ====
<code>
print left("Hello", 2)
</code>
will display
<code>
He
</code>
