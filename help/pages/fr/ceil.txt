===== Ceil =====

==== Format ====

**ceil** ( //expression// )

==== Description ====

REtourne l'entier immédiatement inférieur ou égal à une //expression//.

==== Voir aussi ====

[[floor|Floor]]

==== Exemple ====

<code>
print ceil(9.1)
print ceil(-5.4)
</code>
fera afficher
<code>
10
-5
</code>
