Command Color
Format color colorname
color red, blue, green
Description Sets the current drawing color to colorname or to RGB color red, blue, green. Color names include clear, black, white, red, green, blue, cyan, purple, yellow, orange, gray, darkred, darkgreen, darkblue, darkcyan, darkpurple, darkyellow, darkorange, and darkgray.
