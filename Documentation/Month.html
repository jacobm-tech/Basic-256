Command Month
Format month
Description Returns the current system clock's month. January is 0, February is 1... December is 11.
See Also Day, Hour, Minute, Second, Year
Example 
cls
dim n$(12)
n$ = {"Jan", "Feb", "Mar", "Apr", "May", "Jun", "Jul",
"Aug", "Sep", "Oct", "Nov", "Dec"}
print day + "-" + n$[month] + "-" + year
on New Years will print
1-Jan-2010
