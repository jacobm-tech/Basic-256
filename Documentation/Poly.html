Command Poly
Format poly array, segments
poly {x1, y1, x2, y2, x3, y3 ...}
Description Draws a polygon. The sides of the polygon are defined by the values stored in the array, which should be stored as x,y pairs, sequentially. The number of sides must be given as the second argument.
A polygon can also be specified using a list of x,y pairs enclosed in curly braces {}. If it is defined this way the length parameter and the array variable are not needed.
Example Both of the code blocks below will draw a green triangle on the graphics window.
clg
color green
dim tri{6)
tri = {100, 100, 200, 200, 100, 200}
poly tri, 3

clg
color green
poly {100, 100, 200, 200, 100, 200}
