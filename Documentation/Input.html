Command Input
Format input expression, stringvariable
input expression, numericvariable
input stringvariable
input numericvarvariable
Description Waits for the user to type a line of text into the text output window. When the user hits the enter or return key, the line is read in to stringvar or numericvariable.
User may optionally be prompted for the input by expression.
If a numeric variable is specified and non-numeric data is entered a zero will be assigned to the numeric variable.
Refrences to array elements may also be specified.
