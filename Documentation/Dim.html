Command Dim
Format dim numericvariable( integer )
dim stringvariable$( integer )
Description Returns a newly created array of length integer. Depending on the variable assignment, a string or numerical array is created.
Example 
dim z(5)
z = {1, 2, 3, 4, 5}
print z[0] + " " + z[4]
will print
1 5
dim c$(4)
c$ = {"cow", "brow", "how", "now"}
print c$[2] + " " + c$[3] + " ";
print c$[1] + " " + c$[0] + "?"
will print
how now brown cow?
